# Welcome to DocsSite

This is a clean, minimal documentation site built with SvelteKit and mdsvex.

## Features

- **Clean Design**: Minimal styling focused on content readability
- **Three-Column Layout**: Navigation, content, and table of contents
- **Automatic Navigation**: Generated from your markdown file structure
- **Dark Mode**: Toggle between light and dark themes
- **Table of Contents**: Automatically extracted from page headings

## Getting Started

Explore the documentation using the navigation sidebar on the left. Each page automatically generates a table of contents in the right sidebar for easy navigation within the document.

### Quick Navigation

Use the sidebar to browse through different sections:

- Getting Started guides
- Configuration options
- Advanced features

## Markdown Support

This site supports full markdown syntax with mdsvex integration, including:

- **Bold** and *italic* text
- Code blocks with syntax highlighting
- Lists and tables
- Links and images
